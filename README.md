## 📖 What is claude-rlm?

claude-rlm is an application designed to work with very long text inputs. It uses a method called *recursive sub-agent delegation* to break down and process large pieces of text step-by-step. This allows the program to manage text that would normally be too big to handle all at once.

This software is especially useful if you need to analyze, summarize, or work with large documents or texts but do not want to worry about length limits. The application manages the complexity behind the scenes so you don’t have to.
